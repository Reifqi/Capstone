## CarePet

    Sebuah website yang menyediakan berbagai informasi untuk membantu anda merawat kucing dan anjing kesayangan anda.

## Run Locally

1. Clone the project
    ```bash
      git clone https://github.com/Reifqi/Capstone.git
    ```
    
2. Install dependencies
    ```bash
      npm install
    ```

3. Start the server
    ```bash
      npm run start-dev
    ```

4. Build the project 
    ```bash
      npm run build
    ```
